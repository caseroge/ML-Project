# ML-Project
ML project using MLB data


This is a project designed to explore ML models using pitching data from the MLB. The main purpose is to use ML methods to use the  pitching metrics (spin, velocity, extension, etc) to predict summary metrics like ERA and WHIP. 

First, importing and creating the data set via MLB.com. Then creating the model choice using the xgboost algorithm. I then tested this model against a means moden and a basic decision tree model to validate its use by comparing RMSE, MAE, and R^2. Finally I attempted to tune the xgboost model parameters to dial in its prediction ability

Next, I visualized the model results, as well as comparing the different models.
